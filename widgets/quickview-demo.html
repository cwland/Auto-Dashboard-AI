⚡ Quick View Widgets — Sample Dashboard

An offline preview of the new Quick View widget category. Each card shows ~4–5 of an integration's most important metrics in a compact, dashboard-friendly format. These use the same QuickViewWidget class and per-integration specs the live dashboard uses — here driven by static mock data instead of a network call. Toggle Clickable to see the click-to-open affordance (cards open the integration's address on the real dashboard).

🛡 Pi-hole & AdGuard Home — Sample Dashboard

A fully interactive, offline preview of both widgets. No server — the demo builds the raw API payloads each service returns (Pi-hole v6 /api/stats/summary; AdGuard /control/stats + /control/status + filtering) and runs them through the same DnsHoleApi mapping/computation the live widgets use. Use the buttons to toggle blocking on/off and simulate traffic.

Pi-hole

Uses Pi-hole v6 mapV6() with live percent_blocked.

AdGuard Home

Uses AdGuard computeSummary() (block % derived from blocked/total).

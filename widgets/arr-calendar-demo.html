📺🎬 Sonarr & Radarr Widgets — Sample Dashboard

A fully interactive, offline preview of both widgets. There's no server — the demo generates releases and feeds them through the same Sonarr/Radarr mapping the live widgets use (episode → SxxExx events; movie → one event per release type). Switch between the Upcoming list and the Calendar grid with the toggle in each widget's header, click a highlighted day to see that day's releases, and use the buttons below to change the data. Layout follows Homarr's calendar widget as a reference template.

Sonarr — TV episodes

Try the Calendar tab, then click a day with a dot.

Radarr — movies

Each movie can appear once per release type (cinemas → digital → physical).

🎞 Seerr Widget — Sample Dashboard

A fully interactive, offline preview. No server — the demo builds raw Overseerr/Jellyseerr request objects and runs them through the same SeerrApi.mapRequest / mapAvailability logic the live widget uses. Use the Requests / Stats toggle in the widget header, and the buttons on the right to change the data. Layout follows Homarr's media-requests widgets as a reference template.
